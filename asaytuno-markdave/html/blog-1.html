I Will Make a Missile

Mark Dave Asaytuno • Feb 21, 2026 • 2 min read

I was watching a TikTok live where people were talking about IT students and innovation. They said that as future IT professionals, we should create valuable inventions. Systems that can help people and make a difference.

Then one guy suddenly said:

“Pwede ka gumawa ng system na makakatulong sa ibang bansa, pag gawa ng missile, 'di ba?”

I laughed. A missile? Why would an IT student make a missile? But it made me think. Maybe what he meant was that we should create something powerful. Something impactful. Something that can really change things.

As an IT student, I don't want to build weapons. I want to build systems that help people. Maybe a system for disaster response. Maybe a website for small businesses. Maybe an app that makes daily tasks easier.

We don't need to create something explosive to make an impact. Sometimes, a simple system that works properly is already powerful.

So no, I won't make a missile. But I will build something meaningful. And maybe that's more powerful than a missile.

— Mark Dave
